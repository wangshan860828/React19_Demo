# 项目技术栈
- React 19 + TypeScript + Vite
- 路由管理：React Router V6
- 全局状态管理：Zustand-轻量，简洁，对TypeScript友好
- 服务端状态管理： React Query（TanStack Query）- 用于处理服务端数据状态，支持缓存、重试、分页等功能。
- UI 与样式：TailwindCSS -高效的 Utility-First CSS框架，无需自定义 CSS 即可构建出美观的界面。
- 表单处理：React Hook Form + Zod - 兼顾性能与运行时状态校验。
- 组件库：shadcn/ui - 基于 TailwindCSS 的组件库，提供灵活、可定制、易于接入的组件库。
- 拖拽库：dnd-kit - 专为react设计的现代化、可访问性高的拖拽库，支持触摸设备、鼠标设备、键盘操作等。
